imagenes:

usando la etiqueta "img" puedes colocar imagenes de todo tipo insertando el nombre y formato del archivo. añadiendo el aaprtado "alt=" puedes darle un nombre a la imagen

El tamaño de la imagen puedes ajustarlo con "width=" para lo largo y "height=" para la altura

si deseas alinearla hacia algun lado puedes usar la misma etiqueta que la alineacion de textos que puedes encontrar en la pagina principal

Reproductor de video:

para insertar videos en tu página de html usa la etiqueta "source" y añade entre las comillas el archivo directo del video que deseas reproducir, posterior a eso inserta la etiqueta "type" y escribe el tipo de archivo. Todo esto entre la etiqueta "video" en la cual podras editar el tamaño del video con las mismas funciones que las de las imagenes "width" y "height"
